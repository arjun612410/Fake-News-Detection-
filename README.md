# Fake News Detection using NLP and Machine Learning

This project is a Fake News Detection system built using Natural Language Processing (NLP) and Machine Learning techniques.

The project uses:
- Word2Vec for text vectorization
- Logistic Regression for classification

## Project Workflow
- Data Loading
- Data Cleaning
- Text Preprocessing
- Word2Vec Embedding
- Train-Test Split
- Model Training
- Model Evaluation

## Features
- Detects whether a news article is Fake or Real
- Uses NLP preprocessing techniques such as:
  - Lowercasing
  - Stopword Removal
  - Lemmatization
  - URL Removal
  - HTML Tag Removal

## Model Used
- Logistic Regression

## NLP Technique Used
- Word2Vec

## Accuracy
- Accuracy Score: 89.4%

## Dataset
Dataset Link: https://www.kaggle.com/datasets/studymart/welfake-dataset-for-fake-news
